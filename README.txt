Project Title:
Smart Fitness Activity Tracking & Analysis Library

Description:
This project is a C++ library designed to track and analyze fitness activities.
It allows users to input activities, combine sessions, compare performance,
and calculate calories using function and operator overloading.

Features:
- Function Overloading (3+)
- Operator Overloading (+, -, *, <, ==, <<)
- Real-world fitness tracking system

Sample Output:
Activity: Running
Duration: 30 min
Calories: 150
Distance: 3 km

Combined Activity:
Duration: 75 min
Calories: 450
Distance: 13 km
